# REF-COM-002 — Commercial Excellence

**Status:** Plan aprobado
**Date:** 2024-07-09
**Stage:** XII.5 — Commercial Excellence

## 1. Objetivo

Antes de agregar nuevas funcionalidades (WhatsApp, firma digital, geolocalización), la plataforma debe demostrar que el flujo comercial existente es perfecto. El objetivo no es desarrollar módulos, sino optimizar cada segundo que un asesor vive dentro de Referix.

## 3. Los 8 Sprints

### Sprint 1 — Sales Workflow
Auditar todo el flujo comercial: clics, tiempo, fricción, navegación móvil.
Meta: Una venta debe sentirse natural, no como llenar un formulario.

### Sprint 2 — CERE Excellence
CERE debe actuar como asesor senior: preguntas inteligentes, advertencias, justificaciones, aprendizaje.

### Sprint 3 — Customer Journey
Auditar la experiencia completa del cliente: primer contacto → instalación → seguimiento → upgrade.

### Sprint 4 — Sales Dashboard
Convertir el dashboard en una agenda que responda: ¿Qué debo hacer? ¿Quién necesita seguimiento?

### Sprint 5 — Información Comercial
Revisar todas las tarjetas: TV, GPON, Radio, precio/día, comisión, documentos, cobertura. Cero dudas.

### Sprint 6 — Mobile Excellence
Diseñar primero para móvil, luego escritorio. El asesor trabaja desde el teléfono.

### Sprint 7 — Real Data 100%
Eliminar datos simulados, textos ficticios, KPIs inventados, alertas falsas. Todo desde D1.

### Sprint 8 — Premium Polish
Que Referix se sienta como Stripe, Notion, Linear. Elegante, no solo bonito.
